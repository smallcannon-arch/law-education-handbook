# 法治教育學生手冊

建議 repo 名稱：`law-education-handbook`

## 專案簡介

這是一份適用於三～六年級的法治教育學生手冊，包含閱讀模式、簡報模式與 A4 列印模式，適合作為國小中高年級法治教育教材使用。

## 功能

- 年級切換
- 課程閱讀
- 簡報模式
- 列印目前年級
- 列印全冊
- 手機閱讀

## 技術

- 單一 HTML
- 無後端
- 無外部 CDN
- CSS 與 JavaScript 皆為 inline
- 可直接部署 GitHub Pages
