# 法治教育學生手冊

三～九年級適用的法治教育教材草案整合版，包含閱讀模式、簡報模式與 A4 列印模式。

## 專案簡介

本專案是一份以生活情境為核心的法治教育學生手冊，適用於國小中高年級至國中階段的法治教育、生活教育、班級經營與議題討論。

教材以故事、討論、法律小提醒、行動建議與小測驗，引導學生理解界線、尊重、責任、修復、自我保護與程序概念。網站為單一 HTML 純前端靜態頁，可直接以 GitHub Pages 開啟。

目前版本為三～九年級草案整合版。七～九年級內容仍待後續人工審稿與法規複核；九年級第 7 課〈AI 換臉群組〉目前為「草案大綱／待法規複核」，尚非正式完整故事。

## 主要功能

- 三～九年級教材內容
- 每年級 8 課，共 56 課
- 閱讀模式
- 簡報模式，每年級 56 張投影片
- A4 列印模式
- 手機閱讀版面
- 小測驗依題型顯示作答格式
- 教師參考答案不出現在簡報模式與學生列印版

## 教材架構

- 三年級：我知道界線，也知道可以求助
- 四年級：我看見傷害，也學會負責
- 五年級：我判斷責任，也學會修復
- 六年級：我保護自己，也尊重他人
- 七年級：我在新群體裡，學會界線、站隊與說明
- 八年級：我在數位關係裡，學會尊重、保護與負責
- 九年級：我在社會入口前，學會判斷、責任與重新開始

## 使用方式

1. 開啟 `index.html` 或 GitHub Pages 網頁。
2. 先選擇年級，再選擇課次。
3. 可用閱讀模式帶學生閱讀故事、討論問題、完成小測驗。
4. 可切換簡報模式進行課堂投影與小組討論。
5. 可使用列印目前年級或列印全冊功能，輸出 A4 學生講義。

高風險主題建議由教師帶領，不建議完全放給學生自行閱讀。教師可依班級狀況、學生年齡、事件情節與最新法規調整討論深度。

## 技術特點

- 單一 `index.html`
- 純前端
- 無後端
- 無外部 CDN
- 無外部 script/style/font
- 不使用 `fetch`
- 不使用 `localStorage/sessionStorage`
- 適合 GitHub Pages 靜態部署
