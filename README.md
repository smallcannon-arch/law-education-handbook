# 法治教育學生手冊

三～六年級適用的法治教育教材，包含閱讀模式、簡報模式與 A4 列印模式。

## 專案簡介

本專案是一份通用法治教育學生手冊，適合作為國小中高年級法治教育教材草案。內容以生活情境為核心，協助學生理解界線、尊重、責任、求助與修復等概念。

教材採單一靜態網頁形式，可直接在瀏覽器閱讀，也可切換為課堂簡報或列印為 A4 紙本使用。

## 主要功能

- 三～六年級教材內容
- 每年級 8 課，共 32 課
- 閱讀模式
- 簡報模式，每年級 56 張投影片
- A4 列印模式
- 手機閱讀版面
- 小測驗依題型顯示作答格式
- 教師參考答案不出現在簡報模式與學生列印版

## 教材架構

- 三年級：我知道界線，也知道可以求助
- 四年級：我看見傷害，也學會負責
- 五年級：我判斷責任，也學會修復
- 六年級：我保護自己，也尊重他人

## 使用方式

1. 開啟 `index.html` 即可閱讀教材。
2. 使用年級與課次導覽切換內容。
3. 使用閱讀／簡報模式切換按鈕進入課堂簡報。
4. 使用列印目前年級或列印全冊功能產生 A4 紙本。
5. 手機或平板可直接使用閱讀模式瀏覽教材。

## 技術特點

- 單一 `index.html`
- 純前端
- 無後端
- 無外部 CDN
- 無外部 script/style/font
- 不使用 fetch
- 不使用 localStorage/sessionStorage
- 適合 GitHub Pages 靜態部署
